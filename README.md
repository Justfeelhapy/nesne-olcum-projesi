## Nesne Ölçüm GUI (MATLAB)

Bu proje MATLAB GUI kullanarak görüntü üzerinden nesne ölçümü yapar.

### Çalıştırma
1. MATLAB'ta proje klasörünü açın
2. src/gui_butonlu_nesne_olc.m dosyasını çalıştırın
3. Resim seç butonundan görüntü seçin

### Klasör Yapısı
- src/: MATLAB kodları
- data/: resimler ve ölçüm sonuçları
- figures/: kaydedilen grafikler
